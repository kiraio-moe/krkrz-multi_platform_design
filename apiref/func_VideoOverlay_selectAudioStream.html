VideoOverlay.selectAudioStream

Function/Meaning

[Windows *] Audio stream selection

Type
VideoOverlayclassmethod of
Syntax
selectAudioStream(streamNumber)
Arguments
streamNumber

Specify the audio stream number.

Return value
None (void)
Explanation

Enables the specified audio stream number.
Not available for videos that do not contain multiple audio streams.
On Android it always returns 0 and there is no point in setting it.
.
