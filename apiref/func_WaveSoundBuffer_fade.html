WaveSoundBuffer.fade

Function/Meaning

Start fading

Type
WaveSoundBufferclassmethod of
Syntax
fade(to, time, delay=0)
Arguments
to

Specifies the volume to reach.
See the WaveSoundBuffer.volume property for volume specifications.

time

Specifies the time to fade in ms.

delay

Specifies the wait time in ms to start the fade.

Return value
None (void)
Explanation

Starts a fade (continuous change in volume).
