Storages.extractStoragePath

Function/Meaning

Extraction of storage name path

Type
Storagesclassmethod of
Syntax
extractStoragePath(storage)
Arguments
storage

Specify the storage name from which you want to extract the path.

Return value

The path will be returned. If there is no path, an empty string is returned.

Explanation

Extracts the path part from the specified storage name and returns it.
For example, if you pass "file: //home/dee/hoge.txt", "file: // home / dee /" will be returned.
