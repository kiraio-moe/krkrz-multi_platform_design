VertexBuffer.VertexBuffer

機能/意味

未初期化の頂点バッファを作ります

タイプ
VertexBufferクラスのメソッド
構文
VertexBuffer(size:int, dataType:int, updateType:int, isIndex:bool=false)
引数
size

バッファのバイト数

dataType

データの型。dtByte/dtUByte/dtShort/dtUShort/dtInt/dtFloat

updateType

データ更新頻度。utStream/utStatic/utDynamic

isIndex

インデックスバッファかどうか。頂点バッファならfalse

戻り値
なし (void)
説明
