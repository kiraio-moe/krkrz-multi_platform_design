Dictionary.contains

Function/Meaning

Determines if the specified key exists in the dictionary array.

Type
Dictionaryclassmethod of
Syntax
contains(target:Dictionary, key:string) :bool
Arguments
target

Dictionary array to confirm existence

key

Key to confirm existence

Return value

With or without key

Explanation

Returns true if the key specified in the dictionary exists, false otherwise.
You can omit target if you have context.
It can also be determined by the in operator.
