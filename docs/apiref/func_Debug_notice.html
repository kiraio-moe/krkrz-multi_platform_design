Debug.notice

機能/意味
コンソールへ重要なメッセージを出力
タイプ
Debugクラスのメソッド
構文
notice(message)
引数
message 出力するメッセージを指定します。
戻り値
なし (void)
説明
コンソールへメッセージを出力します。Debug.message と違い、ここで出力したメッセージは ログファイルへの書き出しを途中から開始したとしても、
必ずログファイルに書き出されます ( Debug.message で出力したメッセージは、ある程度さかのぼってまでしか
書き込まれません )。ログファイルを回収したときに有用になるような重要な情報を出力するために使います。
