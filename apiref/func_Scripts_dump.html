Scripts.dump

Function/Meaning

Context dump

Type
Scriptsclassmethod of
Syntax
dump()
Arguments
Return value
None (void)
Explanation

Outputs the details of the contents of each context in the script block currently loaded by TJS2 to a local file.
The disassembled result of the VM code is mainly output.
The output file will be output to the same folder as the Kirikiri executable file, and its file name will be the Kirikiri executable file with .dump.txt added.
