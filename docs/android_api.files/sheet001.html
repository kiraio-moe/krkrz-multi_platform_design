名称メイショウ 概要ガイヨウ Android版バン
コンストラクタ
Window 　 生成セイセイできるウィンドウは1つだけ
メソッド
add 管理オブジェクトの追加 　
bringToFront ウィンドウを最前面に移動 無効ムコウ、何ナニも起オこらない
close ウィンドウを閉じる 　
getMouseVelocity マウス座標移動速度の取得(1.1.0以降) マウスは未対応ミタイオウ
getTouchPoint タッチ座標の取得 　
getTouchVelocity タッチ座標移動速度の取得(1.1.0以降) 　
hideMouseCursor マウスカーソルを一時的に隠す マウスは未対応ミタイオウ
postInputEvent 入力イベントの生成 　
registerMessageReceiver メッセージ受信関数の登録/登録削除 not implemented
remove 管理オブジェクトの削除 　
removeMaskRegion ウィンドウリージョンの解除 無効ムコウ
resetMouseVelocity マウス座標移動速度計測のリセット(1.1.0以降) マウスは未対応ミタイオウ
setInnerSize クライアントサイズの設定 設定セッテイ不可フカ
setMaskRegion ウィンドウリージョンをマスクに従って設定 無効ムコウ
setMaxSize ウィンドウの最大サイズの設定 無効ムコウ
setMinSize ウィンドウの最小サイズの設定 無効ムコウ
setPos ウィンドウ位置の設定 設定セッテイ不可フカ
setSize ウィンドウサイズの設定 設定セッテイ不可フカ
setZoom レイヤ拡大倍率の設定 　
showModal モーダルでウィンドウを表示 モーダルウィンドウは対応タイオウしていない
update ウィンドウ内容の強制的な描画 　
プロパティ
HWND ウィンドウハンドル not implemented
borderStyle ウィンドウ外見 常ツネにbsNone
caption ウィンドウのキャプション Activityのタイトルバー
displayOrientation ディスプレイの向き(1.1.0以降) 　
displayRotate ディスプレイの回転角度(1.1.0以降) 　
drawDevice 描画デバイス 　
enableTouch タッチイベント有効/無効 常ツネに有効ユウコウ
focusable フォーカスを取得可能か 常ツネにtrue
focusedLayer フォーカスを持っているレイヤオブジェクト 　
fullScreen フルスクリーンかどうか 常ツネにフルスクリーン
height ウィンドウの縦幅 Activityの高タカさ(読ヨみ込コみのみ)
hintDelay ヒント表示待ち時間 マウスがないのでヒント無効ムコウ
imeMode デフォルトのIMEモード 無効ムコウ(もう少スコし検討ケントウ)
innerHeight クライアント領域の縦幅 MainViewの幅ハバ(読ヨみ込コみのみ)
innerWidth クライアント領域の横幅 MainViewの高タカさ(読ヨみ込コみのみ)
left ウィンドウの左端位置 常ツネに0
mainWindow メインウィンドウ 　
maxHeight ウィンドウの最大の縦幅 無効ムコウ
maxWidth ウィンドウの最大の横幅 無効ムコウ
minHeight ウィンドウの最小の縦幅 無効ムコウ
minWidth ウィンドウの最小の横幅 無効ムコウ
mouseCursorState マウスカーソル表示状態 マウスは未対応ミタイオウ
primaryLayer プライマリレイヤオブジェクト 　
stayOnTop 常に最上位に表示するかどうか 常ツネにtrue
top ウィンドウの上端位置 常ツネに0
touchPointCount タッチ数 　
touchRotateThreshold マルチタッチ回転閾値 　
touchScaleThreshold マルチタッチ拡大閾値 　
trapKey キー入力をトラップするか 複数フクスウウィンドウはないので常ツネにfalse
useMouseKey マウスキーを使用するかどうか マウスは未対応ミタイオウ
visible ウィンドウが表示されているかどうか 　
waitVSync 垂直同期待ち 無効ムコウ(NDKは7以降イコウでChoreographerをサポート)
width ウィンドウの横幅 Activityの幅ハバ(読ヨみ込コみのみ)
zoomDenom レイヤ拡大倍率(分母) 　
zoomNumer レイヤ拡大倍率(分子) 　
イベント
onActivate ウィンドウがアクティブになったとき 　
onClick ウィンドウがクリックされた マウスは未対応ミタイオウ
onCloseQuery ウィンドウを閉じる確認 　
onDeactivate ウィンドウが非アクティブになったとき 　
onDisplayRotate 画面が回転されたとき(1.1.0以降) 　
onDoubleClick ウィンドウがダブルクリックされた マウスは未対応ミタイオウ
onFileDrop ファイルがドロップされた D&Dはない
onHintChanged ヒントの状態が変化したとき マウスがないのでヒント無効ムコウ
onKeyDown キーが押された 　
onKeyPress 文字が入力された 　
onKeyUp キーが離された 　
onMouseDown マウスのボタンが押された マウスは未対応ミタイオウ
onMouseEnter マウスが入ってきた マウスは未対応ミタイオウ
onMouseLeave マウスが出ていった マウスは未対応ミタイオウ
onMouseMove マウスが移動した マウスは未対応ミタイオウ
onMouseUp マウスのボタンが離された マウスは未対応ミタイオウ
onMouseWheel マウスホイールが回転した マウスは未対応ミタイオウ
onMultiTouch マルチタッチ状態が変化した 　
onPopupHide ポップアップウィンドウを閉じる 無効ムコウ
onResize ウィンドウのサイズが変化した 　
onTouchDown 画面がタッチされた 　
onTouchMove 指が移動した 　
onTouchRotate 回転操作した 　
onTouchScaling 拡大操作した 　
onTouchUp 画面から指が離された
