Window.onKeyPress

機能/意味
文字が入力された
タイプ
Windowクラスのイベント
構文
onKeyPress(key)
引数
key 入力された文字です。
戻り値
なし (void)
説明
文字が入力されたときに発生します。
Window.onKeyDown と異なるのは、onKeyDown が仮想キーコードを扱うのに対し、このイベントは文字そのものを扱います。
押されたキーが文字とは関係のないキー (ファンクションキーなど) の場合はこのイベントは発生しません。
Ctrlキーと同時に押された場合は、以下に示すようなコントロールコードが送られてきます。
