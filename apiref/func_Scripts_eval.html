Scripts.eval

Function/Meaning

Evaluation of the expression

Type
Scriptsclassmethod of
Syntax
eval(expression, name=, linesof=0, context)
Arguments
expression

Specify the expression to be executed as a character string.

name

Specifying the file name for the error message

linesof

Specifying the line number for the error message

context

Specifies the execution context.

Return value

The result of evaluating the expression is returned.

Explanation

Executes the string specified by expression as a TJS2 expression.
Exceptions that occur during script execution are not caught within this method and can be caught by the caller of this method.
