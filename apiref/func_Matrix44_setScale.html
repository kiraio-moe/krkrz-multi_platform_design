Matrix44.setScale

Function/Meaning

Magnification rate setting

Type
Matrix44classmethod of
Syntax
setScale(x:real, y:real, z:real=1)
Arguments
x

X-axis magnification to set

y

Y-axis magnification to set

z

Z-axis magnification to set

Return value
None (void)
Explanation

Set a value for the XYZ expansion position of the matrix.
It is only a setting and does not perform operations with the current matrix.
