Scripts

Scripts クラスは TJS2 に関するメソッドやプロパティを管理します。
このクラスからオブジェクトを作成することはできません。

メンバ

メソッド
compileStorage (ストレージ上のスクリプトのコンパイル )
dump (コンテキストのダンプ )
eval (式の評価 )
evalStorage (ストレージ上の式の評価 )
exec (スクリプトの実行 )
execStorage (ストレージ上のスクリプトの実行 )
getClassNames (クラス名を取得 )
getTraceString (呼び出し履歴の取得 )
setCallMissing (メンバが見つからない時に呼び出されるハンドラの登録 )
プロパティ
textEncoding (スクリプトを読み込む時に使用するテキストエンコーディングを指定 )
イベント
