Dictionary.keys

Function/Meaning

Returns an array of objects' keys.

Type
Dictionaryclassmethod of
Syntax
keys(target:Dictionary) :Array
Arguments
target

An object that counts the number of members

Return value

Array of keys

Explanation

Returns an array of objects' keys. The target works for anything other than dictionary arrays.
The order of the returned arrays is the same as Array.assgin and is not sorted.
