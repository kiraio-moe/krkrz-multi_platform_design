Storages.searchCD

Function/Meaning

[Windows +] Search for CD

Type
Storagesclassmethod of
Syntax
searchCD(volume)
Arguments
volume

Specify the volume label of the CD to search.

Return value

Returns the drive letter of the found CD-ROM drive.

Explanation

Finds the CD drive with the specified volume label and returns the drive letter (such as'H').
Of course, the drive must have the appropriate CD-ROM inserted.
