BitmapLayerTreeOwner.fireTouchRotate

Function/Meaning

Notifies the layer that the rotation operation has been performed

Type
BitmapLayerTreeOwnerclassmethod of
Syntax
fireTouchRotate(startangle, curangle, dist, cx, cy, flag)
Arguments
startangle

Radian angle when multi-touch is started

curangle

Radian angle of touch when an event occurs

dist

Pixel distance of touch when an event occurs

cx

X coordinate of center position (client coordinate)

cy

Center y coordinate (client coordinate)

flag

Multi-touch state flag

Return value
None (void)
