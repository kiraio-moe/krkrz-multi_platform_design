VideoOverlay.setBounds

Function/Meaning

[Windows *] Specify the position and size of the playback rectangle

Type
VideoOverlayclassmethod of
Syntax
setBounds(left, top, width, height)
Arguments
left

Specifies the left edge position of the rectangular area for playing media.

top

Specifies the top position of the rectangular area for playing media.

width

Specifies the width of the rectangular area for playing media.

height

Specifies the vertical width of the rectangular area for playing media.

Return value
None (void)
Explanation

Specifies the position and size of the rectangular area for playing media.
Coordinates are the coordinates within the window's client (the area where the layer can be displayed) and are specified in pixels.
In the current version, this method can only specify the position during layer playback, ignoring the size.
