Storages.getLocalName

Function/Meaning

Get local file name

Type
Storagesclassmethod of
Syntax
getLocalName(path)
Arguments
path

Specify the unified storage name that you want to convert to a local file name.

Return value

The local file name is returned.
If it cannot be converted to a local file name, an empty string is returned or an exception is thrown.

Explanation

Converts the specified unified storage name to the OS native format (local file name) and returns it.
