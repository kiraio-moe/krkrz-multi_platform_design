String.indexOf

Function/Meaning

Returns the position of the substring

Type
Stringclassmethod of
Syntax
indexOf(text:string, offset:int=void) :int
Arguments
text

Substring

offset

Search start position

Return value

If 0 is returned it is the beginning of the string, if -1 is returned it is not found.

Explanation

Searches for text from offset in a string and returns the first found position.
If offset is omitted, the search starts from the beginning of the character string.
