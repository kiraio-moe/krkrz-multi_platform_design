VideoOverlay.setSize

Function/Meaning

[Windows *] Specify the size of the playback rectangle

Type
VideoOverlayclassmethod of
Syntax
setSize(width, height)
Arguments
width

Specifies the width of the rectangular area for playing media.

height

Specifies the vertical width of the rectangular area for playing media.

Return value
None (void)
Explanation

Specifies the size of the rectangular area for playing media in pixels.
In the current version, this method cannot be used during layer playback.
