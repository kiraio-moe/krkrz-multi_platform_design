VideoOverlay.prepare

Function/Meaning

[Windows *] Ready to play

Type
VideoOverlayclassmethod of
Syntax
prepare()
Arguments
Return value
None (void)
Explanation

Draws the first frame of the media on the specified layer and raises the onPeriod event at the end of drawing.
Wait for the onPeriod event before playing after calling the prepare method.

Reference
VideoOverlay.onPeriod
