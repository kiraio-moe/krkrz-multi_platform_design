ShaderProgram.ShaderProgram

Function/Meaning

constructor

Type
ShaderProgramclassmethod of
Syntax
ShaderProgram(vertex:string, fragment:string, vsfile:boolean=true, fsfile:boolean=true)
Arguments
vertex

Specifies the party shader.
When file is true, specify the file name, and when file is false, specify the character string directly.

fragment

Specify the fragment shader.
When file is true, specify the file name, and when file is false, specify the character string directly.

vsfile

Select whether the character string specified by vertex is a file or a direct script specification.

fsfile

Select whether the character string specified by fragment is a file or a direct script specification.

Return value
None (void)
