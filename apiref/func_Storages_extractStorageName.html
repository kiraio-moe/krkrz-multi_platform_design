Storages.extractStorageName

Function/Meaning

Extraction of storage name

Type
Storagesclassmethod of
Syntax
extractStorageName(storage)
Arguments
storage

Specify the storage name from which you want to extract the storage name.

Return value

The storage name is returned.
If there is no storage name, an empty string is returned.

Explanation

Extracts and returns the part of the storage name (excluding the path) from the specified storage name.
For example, if you pass "System / hoge.txt", "hoge.txt" will be returned.
