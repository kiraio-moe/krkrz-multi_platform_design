Offscreen

オフスクリーン。
いわゆるレンダーターゲット。テクスチャとしても利用可能です。
OpenGL ES2.0 では、FBO, Renderbuffer, Texture が生成されます。
レンダーターゲットとして設定しないのであれば、わざわざこのクラスを使用する必要性はありません。
Textureクラスを使用した方が良いです。
