Dictionary.assignStruct

Function/Meaning

Copy the dictionary array

Type
Dictionaryclassmethod of
Syntax
assignStruct(src:Dictionary)
Arguments
src

Copy source dictionary array

Return value
None (void)
Explanation

Copies the entire contents of the other dictionary array specified by the argument.
Unlike the assign method, if a member has an array or dictionary array, it will also recursively copy its contents (in the case of the assign method, it just copies the reference).
