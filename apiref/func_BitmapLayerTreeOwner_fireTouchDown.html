BitmapLayerTreeOwner.fireTouchDown

Function/Meaning

Notify the layer that it has been touched

Type
BitmapLayerTreeOwnerclassmethod of
Syntax
fireTouchDown(x, y, cx, cy, id)
Arguments
x

X coordinate of the touched position (client coordinate)

y

Y coordinate of the touched position (client coordinate)

cx

Number of horizontal pixels that the finger is in contact with

cy

Number of vertical pixels that the finger is in contact with

id

Touch ID

Return value
None (void)
