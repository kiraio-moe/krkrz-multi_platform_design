ImageFunction

ImageFunction クラスは、画像を操作するクラスです。
このクラスからオブジェクトを作成することはできません。

メンバ

メソッド
adjustGamma (ガンマ補正 )
colorRect (矩形半透明塗りつぶし )
copy9Patch (9 path を利用した画像コピー(マルチプラットフォーム版以降) )
doBoxBlur (矩形ブラーをかける )
doGrayScale (グレースケール変換 )
drawGlyph (文字描画 )
drawText (文字描画 )
fillRect (矩形塗りつぶし )
flipLR (左右反転 )
flipUD (上下反転 )
operateAffine (アフィン変換演算合成 )
operateRect (矩形演算合成 )
operateStretch (拡大縮小演算合成 )
