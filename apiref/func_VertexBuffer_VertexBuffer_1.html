VertexBuffer.VertexBuffer

Create an uninitialized vertex buffer

Type
VertexBufferclassmethod of
Syntax
VertexBuffer(size:int, dataType:int, updateType:int, isIndex:bool=false)
Arguments
size

Number of bytes in the buffer

dataType

Data type. dtByte / dtUByte / dtShort / dtUShort / dtInt / dtFloat

updateType

Data update frequency. utStream / utStatic / utDynamic

isIndex

Whether it is an index buffer. False for vertex buffer

None (void)
