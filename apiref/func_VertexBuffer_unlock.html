VertexBuffer.unlock

Function/Meaning

Unlock vertex data

Type
VertexBufferclassmethod of
Syntax
unlock()
Arguments
Return value
None (void)
Explanation

For plugins (only available with OpenGL ES 3.0 or later).
When the data has been passed, call it before the buffer is used.
It is preferable not to keep the lock, but to unlock it immediately after updating.
