VideoOverlay.setPeriodEvent

Function/Meaning

[Windows *] Specifying the occurrence of an event at a specified frame

Type
VideoOverlayclassmethod of
Syntax
setPeriodEvent(eventFrame)
Arguments
eventFrame

Specifies the frame that raises the onPeriod event.

Return value
None (void)
Explanation

Raises the onPeriod event at the specified frame.
The onPeriod event is cleared once it occurs.
If you want to generate it again, set it again with this method.
