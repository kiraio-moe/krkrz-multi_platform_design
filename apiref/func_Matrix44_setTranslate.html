Matrix44.setTranslate

Function/Meaning

Movement amount setting

Type
Matrix44classmethod of
Syntax
setTranslate(x:real, y:real, z:real=0)
Arguments
x

X coordinate value to set

y

Y coordinate value to set

z

Z coordinate value to set

Return value
None (void)
Explanation

Set a value for the XYZ movement position of the matrix.
It is only a setting and does not perform operations with the current matrix.
