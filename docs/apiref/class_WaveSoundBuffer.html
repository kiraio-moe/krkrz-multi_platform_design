WaveSoundBuffer

WaveSoundBuffer クラスは、PCMの再生を管理するクラスです。
WaveSoundBuffer クラスでは、ループチューナ で作成した .sli ファイルを読み込み、処理することができます。
詳しくはループチューナの説明をご覧ください。
