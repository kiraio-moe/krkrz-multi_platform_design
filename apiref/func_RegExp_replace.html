RegExp.replace

Function/Meaning

Replaces a string and returns the string after the replacement

Type
RegExpclassmethod of
Syntax
replace(text:string, replace:string) :string
Arguments
text

Target character string

replace

Replacement string

Return value

Matches the target character string, replaces the matched part with the replacement character string, and returns the character string after the replacement.

Explanation

If you pass a function as the replacement string instead of a string, that function will be called for the replacement operation.
The function takes one argument and is passed an array object that represents the matched part of the target string (see the exec method for this array).
The matched part in the target string is replaced with the string returned by the function.
This method ignores the start property.
