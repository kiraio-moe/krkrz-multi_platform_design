Clipboard.hasFormat

Function/Meaning

[Windows *] Does the specified format exist in the clipboard?

Type
Clipboardclassmethod of
Syntax
hasFormat(format)
Arguments
format

Specify the format you want to check for existence.
In the current version, you can only specify cbfText, which allows you to check if there is textual data in the clipboard.

Return value

True if the specified format exists in the clipboard, false if it does not exist

Explanation

You can check if the specified format exists in the clipboard.
The OS-provided clipboard can have multiple formats at the same time, so you can use this method to find out if there is a particular format in it.
Android always returns false.
