Bitmap.Bitmap

Function/Meaning

Building a sized Bitmap object

Type
Bitmapclassmethod of
Syntax
Bitmap(width, height, bpp=32)
Arguments
width

Specifies the width of the image.

height

Specifies the height of the image.

bpp

Specifies the number of bits for each pixel in the image. Specify either 32 or 8.

Return value
None (void)
Explanation

Builds an object of class Bitmap with the specified size.
