Layer.bringToBack

Function/Meaning

Move to the back

Type
Layerclassmethod of
Syntax
bringToBack()
Arguments
Return value
None (void)
Explanation

In the stacking order, it moves to the back of the sibling layers (layers with the same parent).
Executing this method sets the Layer.absoluteOrderMode property of the parent layer to false.
