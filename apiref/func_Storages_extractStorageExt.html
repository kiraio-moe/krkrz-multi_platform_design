Storages.extractStorageExt

Function/Meaning

Extraction of storage name extension

Type
Storagesclassmethod of
Syntax
extractStorageExt(storage)
Arguments
storage

Specify the storage name from which you want to extract the extension part.

Return value

The extension part is returned. The extension part also includes. (Dot). If there is no extension, an empty string is returned.

Explanation

Extracts the extension part from the specified storage name and returns it.
