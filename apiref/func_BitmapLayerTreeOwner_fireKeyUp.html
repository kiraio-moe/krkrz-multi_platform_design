BitmapLayerTreeOwner.fireKeyUp

Function/Meaning

Notifies the layer that the key has been released

Type
BitmapLayerTreeOwnerclassmethod of
Syntax
fireKeyUp(key, shift)
Arguments
key

Virtual key code for the released key

shift

The state of shift keys and mouse buttons that were pressed at the same time when the keys were released

None (void)
