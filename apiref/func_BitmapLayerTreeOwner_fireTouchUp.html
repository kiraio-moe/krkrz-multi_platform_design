BitmapLayerTreeOwner.fireTouchUp

Function/Meaning

Notifies the layer that the touch has been released

Type
BitmapLayerTreeOwnerclassmethod of
Syntax
fireTouchUp(x, y, cx, cy, id)
Arguments
x

X coordinate of the separated position (client coordinate)

y

Y coordinate of the separated position (client coordinate)

cx

Number of horizontal pixels that the finger is in contact with

cy

Number of vertical pixels that the finger is in contact with

id

Touch ID

Return value
None (void)
Explanation
