Plugins.unlink

Function/Meaning

Detaching the plugin

Type
Pluginsclassmethod of
Syntax
unlink(name)
Arguments
name

Specify the plug-in to disconnect.

Return value

True if the disconnection succeeds, false if it fails

Explanation

Detach the specified plug-in.
Some plugins do not respond to release (for example, when the object is in use).
If it fails, it will return false.
