Array.split

機能/意味

文字列を分割します。

タイプ
Arrayクラスのメソッド
構文
split(pattern:string, text:string, reserve, ignoreSpace:bool=false) :Array
引数
pattern

パターンまたはデリミタ集合を指定します。
パターンは正規表現オブジェクトである必要があります。
デリミタ集合には、区切り記号となる文字を指定します。
文字列を指定することで区切り文字を複数指定できます。

text

分割する文字列を指定します。

reserve

将来のバージョンのために予約されています。

ignoreSpace

空の要素 ( たとえばカンマをデリミタにしているときに ,,,,, など ) の場合にその要素を配列にコピーしない場合は true を指定します。

戻り値

この Array クラスのオブジェクト自身を返します。

説明

配列を初期化し、この配列に、文字列をパターンまたはデリミタ集合で区切った物をコピーします。
