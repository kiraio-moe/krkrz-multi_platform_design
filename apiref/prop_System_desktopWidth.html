System.desktopWidth

Function/Meaning
Desktop width
Type
System Class Properties
Explanation

Represents the width of the desktop (the area in which the window can be displayed) in pixels.
The value is for a display with a main window.
If there is no main window, the primary display is the target.
Screen width on Android.
