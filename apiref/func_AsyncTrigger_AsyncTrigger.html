AsyncTrigger.AsyncTrigger

Function/Meaning

Building an AsyncTrigger object

Type
AsyncTriggerclassmethod of
Syntax
AsyncTrigger(owner, actionname=action)
Arguments
owner

Specify where the event will occur.

actionname

Specifies the method name that receives the event in the event destination object specified by owner.
If you specify an empty string, the owner is considered a method and the owner is called directly each time an event occurs.

Return value
None (void)
Explanation

Build an object of class AsyncTrigger.
