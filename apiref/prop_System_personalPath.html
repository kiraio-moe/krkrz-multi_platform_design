System.personalPath

Function/Meaning
My document path
Type
System Class Properties
Explanation

Represents the path of the user's My Documents.
For Windows, the folder represented by Personal in HKEYCURRENTUSER \ Software \ Microsoft \ Windows \ CurrentVersion \ Explorer \ Shell Folders in the registry is returned.
This usually refers to the "My Documents" folder.
If this folder does not exist, it returns the same folder as System.exePath.
