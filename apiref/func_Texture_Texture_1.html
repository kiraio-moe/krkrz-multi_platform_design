Texture.Texture

Function/Meaning

Mipmap generated constructor

Type
Textureclassmethod of
Syntax
Texture(sizelist:Array, filename:string, type=stFastAreaAvg, typeopt:real=1)
Arguments
sizelist

Specify the size to generate the mipmap by the combination of width and height in the array.

filename

Image file name

type

Specifies the reduction algorithm. Same as the type argument for ImageFunction.operateStretch or the type argument for Layer.operateStretch.

typeopt

Sharpness during 3D interpolation. It currently has no meaning with other interpolation methods.

Return value
None (void)
