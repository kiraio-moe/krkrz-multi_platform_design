Bitmap.setSize

Function/Meaning

Image size setting

Type
Bitmapclassmethod of
Syntax
setSize(width, height)
Arguments
width

Specifies the width of the image in pixels. This value can also be obtained and set with the Bitmap.width property.

height

Specifies the vertical width of the image in pixels. This value can also be obtained and set with the Bitmap.height property.

Return value
None (void)
Explanation

Specify the image size.
