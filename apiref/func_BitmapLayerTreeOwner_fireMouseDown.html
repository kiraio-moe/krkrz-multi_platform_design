BitmapLayerTreeOwner.fireMouseDown

Function/Meaning

Notify the layer of mouse presses

Type
BitmapLayerTreeOwnerclassmethod of
Syntax
fireMouseDown(x, y, mb, flags)
Arguments
x

X coordinate of the position where the mouse button was pressed (client coordinate)

y

The y coordinate of the position where the mouse button was pressed (client coordinate)

mb

Pressed mouse button

flag

The state of shift keys that were pressed at the same time when the mouse button was pressed

Return value
None (void)
Explanation
