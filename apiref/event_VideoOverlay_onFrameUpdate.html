VideoOverlay.onFrameUpdate

Function/Meaning

[Windows *] Video frame updated

Type
VideoOverlayclassEvent
Syntax
onFrameUpdate(frame)
Arguments
frame

Video frame number

Return value
None (void)
Explanation

This method is called after the video frame has been updated.
The argument frame does not always exactly match the currently displayed video frame.
Only available in layer drawing mode and mixer mode.
It does not occur on Android.
