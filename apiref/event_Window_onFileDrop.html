Window.onFileDrop

Function/Meaning

[Windows *] file dropped

Type
WindowclassEvent
Syntax
onFileDrop(files)
Arguments
files

An Array object that contains the dropped file names.

Return value
None (void)
Explanation

Occurs when a file is dropped into a window from Explorer etc.
An array object is passed as an argument even if a single file is dropped (the first element will be that file).
