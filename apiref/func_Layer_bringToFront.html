Layer.bringToFront

Function/Meaning

Move to the front

Type
Layerclassmethod of
Syntax
bringToFront()
Arguments
Return value
None (void)
Explanation

In the stacking order, it moves to the front of the sibling layers (layers with the same parent).
Executing this method sets the Layer.absoluteOrderMode property of the parent layer to false.
