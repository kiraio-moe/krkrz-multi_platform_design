BitmapLayerTreeOwner.fireRecheckInputState

Function/Meaning

Call every 1 second if necessary.

Type
BitmapLayerTreeOwnerclassmethod of
Syntax
fireRecheckInputState()
Arguments
Return value
None (void)
Explanation

The current layer's mouse position and hint updates, cursor type, and layer Enter / Leave are rechecked.
Internally, the mouse cursor movement 0 call is made.
