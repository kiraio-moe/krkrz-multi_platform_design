ImageFunction.flipLR

Function/Meaning

Flip horizontal

Type
ImageFunctionclassmethod of
Syntax
flipLR(bmp, rect=null)
Arguments
bmp

Specifies the Bitmap object to invert.

rect

The area to be inverted is specified by the Rect object (at the image position of the Bitmap).
When not specified, the entire target is applicable.

Return value
None (void)
Explanation

Flip the image horizontally.
