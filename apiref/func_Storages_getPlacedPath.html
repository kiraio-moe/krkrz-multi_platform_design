Storages.getPlacedPath

Function/Meaning

Search for storage

Type
Storagesclassmethod of
Syntax
getPlacedPath(storage)
Arguments
storage

Specify the storage name you want to search.

Return value

The found location is returned with a unified storage name.
If not found, an empty string is returned.

Explanation

Searches the storage specified by storage from the automatic search path.
